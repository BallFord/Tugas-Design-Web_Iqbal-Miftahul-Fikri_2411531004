Website ini dibikin dengan tujuan untuk menyelesaikan tugas pertama saya dalam mata kuliah Desain Web. Pada mata kuliah ini, saya diberi kesempatan untuk memahami dasar-dasar pembuatan halaman web, mulai dari struktur dalam penggunaan HTML sampai dengan pengaturan tampilan dengan CSS.

Di website ini terdapat link di navbar bernama About yang berisi informasi tentang diri saya, serta link Contact yang memuat detail kontak seperti email, nomor telepon, alamat, dan akun media sosial.

Saat ini, website telah ditingkatkan menjadi sebuah Progressive Web App (PWA). Ini berarti Anda bisa meng-install halaman ini di perangkat Anda untuk akses yang lebih cepat dan bahkan bisa diakses saat offline.
